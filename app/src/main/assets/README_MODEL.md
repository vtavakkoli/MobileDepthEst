# Model asset folder

The app can download `da3_small_gpu_fp16.tflite` automatically on first use from:

https://huggingface.co/litert-community/Depth-Anything-3-Small/resolve/main/da3_small_gpu_fp16.tflite

For fully offline APK testing, run `scripts/download_model.sh` or place the model here manually before building.
